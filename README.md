# calendar-app
A simple calendar that you can set yours reminders.

Created with:
```
node v16.13.2
npm 8.1.2
```

### Note
To get weather forecast, put your API key from [OpenWeather](https://openweathermap.org/) in VUE_APP_OPEN_WEATHER_KEY (you can remove the single quotes).

## Project setup
```
npm install
```
